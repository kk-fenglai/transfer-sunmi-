📚 API Documentation Information

  • Endpoint: POST /v2/mdm/open/open/deviceCenter/partner/getSubAccountList
  • Authentication: HMAC-SHA256 signature required
  • Payload: Empty JSON object {}
  • Response: List of sub-account entities with entity_id and entity_name
  • Success Code: 1
  • Error Codes: 13010001 (Server Error), 13010002 (Parameter Error), 30001 (Cannot get ability)
